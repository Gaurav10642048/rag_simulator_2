🔍 RAG Simulator

Understand how Retrieval-Augmented Generation (RAG) works step-by-step. This simulator demonstrates knowledge ingestion, chunking, embeddings, semantic search, prompt augmentation, and grounded LLM response generation.

📚 RAG Pipeline

Step 1 — Knowledge Base Upload and clean documents
Step 2 — Chunking Break text into smaller chunks
Step 3 — Embeddings Convert chunks into vectors
Step 4 — Query Embedding Convert user query into vector
Step 5 — Similarity Search Find semantically relevant chunks
Step 6 — Prompt Augmentation Inject retrieved context
Step 7 — LLM Answer Generate grounded response
